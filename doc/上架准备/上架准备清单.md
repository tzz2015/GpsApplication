# 遨游轨迹 · 上架准备清单

使用说明：按顺序逐项完成，在 `[ ]` 中填入 `x` 表示已完成。

---

## 1. 账号与认证

- [ ] 已在华为开发者联盟注册账号
- [ ] 已完成实名认证（个人/企业）
- [ ] 可正常登录 AppGallery Connect（AGC）

## 2. 应用与签名

- [ ] 在 AGC 创建 HarmonyOS 应用，包名与工程一致
- [ ] 在 DevEco Studio 生成密钥库（.p12）与证书请求（.csr）
- [ ] 在 AGC 申请并下载发布证书（.cer）
- [ ] 在 AGC 申请并下载发布 Profile（.p7b）
- [ ] 在项目中配置签名（Store file / Certpath / Profile）
- [ ] 成功构建已签名的 .app 包
- [ ] 在真机安装并验证功能、权限正常

## 3. 应用市场材料

- [ ] 应用图标（符合规范尺寸）
- [ ] 应用名称、简介、详细描述文案定稿
- [ ] 应用截图（至少 3–5 张，主流程页面）
- [ ] 演示视频（可选）
- [ ] 关键词、分类已确定
- [ ] 隐私政策链接（可公网访问）
- [ ] 用户协议链接（可公网访问）

## 4. 合规与说明

- [ ] 隐私政策内容与实际上报/收集行为一致
- [ ] 用户协议已审核
- [ ] 权限说明已准备（应用内或应用市场描述）

## 5. 提交与发布

- [ ] 在 AGC 应用发布页填写全部必填项
- [ ] 上传已签名的 .app 包
- [ ] 提交审核
- [ ] 关注审核结果，按反馈修改后必要时重新提交
- [ ] 审核通过后执行上架

---

## 材料存放建议

可将以下材料放在本目录或子目录，便于统一管理：

- `图标与截图/`：应用图标、应用市场截图
- `签名与证书/`：.p12、.cer、.p7b（注意 .p12 需安全保管，勿提交到公开仓库）
- `文案/`：应用名称、简介、描述、关键词等定稿文案

*清单版本：v1.0*
