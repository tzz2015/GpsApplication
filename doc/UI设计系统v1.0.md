# 遨游轨迹 App UI 设计系统 v1.0（已由 v2.0 替代）

> **当前设计请以 [UI设计系统v2.0.md](UI设计系统v2.0.md)（传承·山林）为准。** 下文为 v1.0 历史参考。  
> 基于 UI-UX-Pro-Max Skill 与 HarmonyOS 设计规范；以原型为单一真相源。

---

## 一、设计原则

1. **HarmonyOS 风格**：界面简洁、信息层级清晰，避免多余装饰。
2. **单一真相源**：所有 UI 以 `doc/prototype/prototype.html` 为准，ArkTS 实现与之对齐。
3. **地图与底部控制**：轨迹记录/详情页采用「上方沉浸式地图 + 下方圆角控制面板」，底部面板圆角明显并预留安全区。
4. **登录与启动**：启动页深蓝渐变 + 居中 Logo/标语；登录页同系渐变，登录按钮使用华为官方红色与规范文案。

## 三、字体与排版

- **字体族**：`HarmonyOS Sans`, `PingFang SC`, `-apple-system`, `sans-serif`
- **基础字号**：14px，行高 1.5

| 用途 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 大标题（如首页标题） | 24px | 800 | text-primary |
| 页面标题 | 18px | 700 | text-primary |
| 卡片标题/列表主信息 | 16px | 700 | text-primary |
| 正文 | 15px | 400 | text-primary / secondary |
| 说明/提示 | 13px | 400 | text-secondary |
| 辅助/标签 | 12px | 500 | text-hint |
| 小标签（badge） | 11px | 600 | 依语义 |

---

## 四、间距与圆角

- **间距基准（8px 网格）**：4, 8, 12, 16, 20, 24, 32, 40, 48
- **安全区**：`--safe-top: 44px`，`--safe-bottom: 34px`，底部 Tab 与底部面板需加 `padding-bottom: var(--safe-bottom)`

| 圆角用途 | 变量/值 | 说明 |
|----------|---------|------|
| 小控件 | `--radius-sm: 8px` | 输入框、小按钮 |
| 默认 | `--radius: 12px` | 卡片、按钮、统计块 |
| 大卡片/面板 | `--radius-lg: 16px` | 底部录制面板、弹窗顶部 |
| 超大/胶囊 | `--radius-xl: 20px` | 主按钮、大卡片 |
| 全圆 | `--radius-full: 999px` | 头像、标签、胶囊按钮 |

---

## 五、阴影

- **卡片**：`0 1px 3px rgba(0,0,0,0.06)`
- **浮起**：`0 4px 12px rgba(0,0,0,0.08)`
- **弹窗/底部面板**：`0 -4px 24px rgba(0,0,0,0.1)` 或 `0 8px 32px rgba(0,0,0,0.12)`

---

## 六、核心组件约定

- **主按钮**：背景 `primary` 或 `accent`（新建/开始用 accent），圆角 24px，高度约 50px，白字。
- **次要按钮**：白底 + 主色描边，或灰底。
- **新建轨迹入口**：白底 + 虚线描边 `accent`，文字与图标 `accent`。
- **Tab 选中**：主色文字 + 底部 3px 主色条，宽度约 24px。
- **Badge**：成功/警告等语义背景 + 语义色文字，小圆角胶囊。
- **底部 TabBar**：白底、顶部分割线、未选灰色、选中主色，高度 56px + safe-bottom。

---

## 七、页面级说明

- **P0-1 启动页**：深蓝渐变全屏，Logo 80–88px 圆角方，应用名 + 标语，底部「点击进入」可选。
- **P0-2 登录页**：同系蓝渐变，Logo + 标题 + 标语，华为红按钮「华为账号一键登录」，底部协议勾选 + 安全区。
- **P1-0 首页**：白顶栏 + 双 Tab（我的轨迹 | 公开轨迹），统计卡片用主色渐变，创建按钮用 accent 虚线样式，轨迹卡片白底 + 顶部淡绿地图占位 + 橙色轨迹线。
- **P1-3 轨迹记录**：全屏地图（淡绿/地图风格占位），右上角图层/3D/定位控件，**底部圆角白面板**（上圆角 16–20px），内含录制状态、里程/用时/海拔、操作按钮，底部留 safe-bottom。
- **P1-4 轨迹详情**：顶部地图条 + 标题/统计/打卡点时间轴，操作按钮与首页风格一致。
- **P3-1 个人中心**：**头部深灰蓝渐变**（#334155 → #1E293B），头像 + 昵称 + 统计条，下方白底菜单列表。

---

## 八、与 ArkTS 的对应

- 颜色：在 `entry/src/main/resources/base/element/color.json` 中保持与上述色值一致（已有项可替换为新值）。
- 圆角、间距、字号：在各自页面 `.ets` 中按本设计系统数值书写，或通过资源引用。
- 原型更新后，ArkTS 以 `doc/prototype/prototype.html` 为视觉基准做对照修改。
